Vypište výběhy (všechny sloupce) tak, že druh oplocení bude anglicky: dřevo=wood, klec=cage, sklo=glass, pletivo=wire net.

SELECT idVybehu, rozloha, CASE WHEN oploceni = ‘drevo‘ THEN ‘wood‘ WHEN oploceni = ‘klec‘ THEN ‘cage‘ WHEN oploceni = ‘sklo‘ THEN ‘glass‘ WHEN oploceni = ‘pletivo‘ THEN ‘wire net‘ END FROM tvybeh;

Kolik exemplářů (kusů) zvířat bydlí ve výbězích s lichým číslem?

SELECT COUNT(*) FROM texemplar JOIN tvybeh ON texemplar.idVybehu = tvybeh. idVybehu WHERE texemplar.idVybehu % 2 <> 0;

Jaký je rozdíl mezi největší a průměrnou rozlohou výběhů?

SELECT (MAX(rozloha) – AVG (rozloha))  as max FROM tvybeh;

Vypište živočišný druh (latinská i česká jména) s nejdelším latinským názvem (součet délek rodového a druhového jména).

SELECT rodl, druhl, rodc, druhc FROM tdruh GROUP BY rodl, druhl ORDER BY MAX (char_length(rodl) + char_length(druhl)) DESC LIMIT 1;

K tabulce Tdruh přidejte sloupec Pismeno datového typu char(1). V tomto sloupci nechť jsou počáteční písmena latinských rodových jmen zvířat.

ALTER TABLE tdruh ADD COLUMN pismeno CHAR(1); UPDATE tdruh SET pismeno = LEFT (rodl, 1);

Z rozloh výběhů spočítejte průměr a směrodatnou odchylku a vypište interval <průměr-směrodatná odchylka, průměr+směrodatná odchylka>. Směrodatná odchylka se anglicky řekne standard deviation.

SELECT (AVG(rozloha) – STDDEV_POP(rozloha)) AS dolnimez, (AVG(rozloha) + STTDEV_POP(rozloha)) AS hornimez FROM tvybeh;

Jaký je rozdíl mezi nejvyšší a nejnižší teplotou v pavilonech?

SELECT (MAX(teplota) – MIN(teplota)) AS max FROM tpavilon;

Vypište všechny sloupce pro výběh s pátou nejmenší rozlohou.

SELECT * FROM tvybeh ORDER BY rozloha ASC LIMIT 1 OFFSET 4;

Jakou průměrnou rozlohu mají výběhy oplocené dřevem?

SELECT AVG(rozloha) FROM tvybeh WHERE oploceni = ‘drevo‘;

Kolik je výběhů oplocených dřevem nebo sklem?

SELECT COUNT(*) FROM tvybeh WHERE oploceni = ‘drevo‘ OR oploceni = ‘sklo‘;

Vypište id prázdných výběhů.

SELECT tvybeh.idVybehu FROM tvybeh LEFT JOIN tdruh_vybeh ON tvybeh.idVybehu = tdruh_vybeh.idVybehu WHERE tdruh_vybeh.idVybehu IS NULL;

Vypište prázdné pavilony (všechny sloupce).

SELECT tpavilon.idPavilonu, tpavilon.teplota, tpavilon.osvetleni, tpavilon.rozloha FROM tpavilon LEFT JOIN tdruh_pavilon ON tpavilon.idPavilonu = tdruh_pavilon.idPavilonu WHERE tdruh_pavilon.idPavilonu IS NULL;

Jaká je největší hodnota osvětlení u pavilonů s rozlohou menší než 100m?

SELECT MAX(osvetleni) FROM tpavilon WHERE rozloha < 100;

Vypište první písmena latinských rodových jmen a u každého, kolikrát se vyskytuje v databázi druhů. Seřaďte podle abecedy.

SELECT DISTINCT LEFT(rodl, 1) AS pismeno, COUNT(*) FROM tdruh GROUP BY pismeno ORDER BY pismeno;

Jaká je největší délka českého druhového jména zvířat?

SELECT MAX(druhc) FROM tdruh;

Pro každý živočišný druh žijící v ZOO vypište jeho česká jména a kolik je v ZOO jeho exemplářů.

SELECT tdruh.rodc, tdruh.druhc, COUNT(*) AS pocet FROM tdruh JOIN Texemplar ON tdruh.rodl=texemplar.rodl AND Ttdruh.druhl=texemplar.druhl GROUP BY tdruh.rodc, tdruh.druhc ORDER BY tdruh.rodc, tdruh.druhc ASC;

Vypište latinské rody, které mají více druhů než 3.

SELECT tdruh.rodl, COUNT(*) FROM tdruh GROUP BY tdruh.rodl HAVING COUNT(druhl) > 3;

Kolik exemplářů (kusů) zvířat aktuálně žije v pavilonech s teplotou nad 25 stupňů?

SELECT COUNT(*) FROM Tpavilon JOIN TExemplar ON tpavilon.idPavilonu = texemplar.idPavilonu WHERE teplota > 25 AND texemplar.idPavilonu IS NOT NULL;
